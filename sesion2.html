Métodos para vincular reglas css al documento

Existen 3 métodos para vincular estilos a la estructura html. Son los siguientes:

  1. Mediante el atributo style
  2. Mediante la etiqueta style
  3. Mediante un documento .css

Tipos de selectores CSS:

Existen 3 selectores diferentes para aplicar css a los elementos html:

  • Selector de etiqueta (ejemplo: <cite>)
  • Selector de id (ejemplo: #container)
  • Selector de clase (ejemplo: .alerta)

El selector de etiqueta aplica las reglas css a todas las etiquetas seleccionadas

El selector de id aplica las reglas css al elemento que tenga ese id (No puede haber dos elementos con el mismo id)

El selector clase aplica las reglas css a TODOS los elementos html que tengan el atributo "class" con el nombre de ese selector
